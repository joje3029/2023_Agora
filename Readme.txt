요구사항 : 시나리오 AI를 할까했으나(어제) 그보다는 메인 기능인 토론방이 더 급하다고 밤새 생각이 바뀜.
즉, 토론방을 합니다.

1. 우선 토론방을 누르면 칼럼 처럼 토론방 제작된거 리스트가 뜨는 페이지가 만들어져야함.
=칼럼 리스트 참조해서 로직 변경

쌤은 라디오버튼을 write에 사용하심.
이걸 활용하면

- > 토론방을 만든다는 원리는 같음.
그럼 1(채팅) / 2(토론)
이냐에 따라 어느쪽으로 갈지 정해주면 되겠네. 아!

만드는 폼에서 넘어올꺼
토론방 이름(roomName)이랑 type

병호님 채팅방 로직 분석

홈 에서 채팅방 목록 클릭 -> 모달로 채팅방 목록이 뜸.

- 채팅방 생성을 눌렀을 때
-> 채팅방생성 : 제목, 최대인원수, 공개여부(비공개면 비번설정) 후 생성버튼
-> 채팅 형태 나옴. => 카톡 채팅과 형태 비슷
- 방을 눌렀을 때
-> 채팅방 입장했다는 말과함께 채팅 => 카톡 채팅이랑 비슷하네.
